name: icloud-help
description: Explain how to use the icloud-mail-manager MCP tools safely and effectively.
---

When the user asks how to use iCloud mail tools, provide concise guidance and suggest the correct MCP tool among:
- list_folders
- manage_folder
- search_emails
- read_email
- manage_attachments
- send_email
- organize_email
- suggest_organization

Always remind the user to configure credentials through environment variables instead of hardcoding secrets.
